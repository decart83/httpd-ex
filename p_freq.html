Letter Frequency

Overview

This letter frequency dashboard was created for a group of students working on a cryptography assgnment. Their task was to crack the code and decrypt the secret message.

Goals

Compute the conditional probability of a letter based on word length

Approach

A list of English words was downloaded from the internet. Then, I used Tableau public to develop and host the dashboard.

Results

This dashboard allowed them to see patterns in the code quickly by using the conditional probability of a letter based on word length.
